- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 三、功能描述
系统分为（身份）：学生、教师、管理员。

**学生功能：**
1. 登录
2. 注册
3. 心理健康学习
4. 试卷列表
5. 公告通知
6. 留言反馈
7. 个人中心
8. 辅导预约
9. 考试管理

**教师功能：**
1. 登录
2. 注册
3. 个人中心
4. 辅导预约管理
5. 学生信息管理
6. 测评结果分析管理
7. 心理健康学习管理
8. 试卷管理
9. 试题管理
10. 留言板管理
11. 考试管理



**管理员功能：**
1. 登录
